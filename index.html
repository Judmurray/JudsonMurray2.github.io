Transfer Learning Image Classification with PyTorch

This is a project I completed in Graduate studies leveraging the "Animals-10" dataset from Kaggle, containing 26,179 images across 10 animal classes, to build an image classification model using PyTorch. The project involves translating Italian labels to English and implementing two model architectures: a traditional Convolutional Neural Network (CNN) using keras and PyTorch, and a hybrid model combining a pretrained foundation model with a Transformer Encoder head. The aim was to compare the performance of these models in classifying animals into categories like sheep, horse, cow, dog, cat, squirrel, spider, chicken, elephant, and butterfly. This project demonstrates the application of advanced deep learning techniques and hybrid models for improved image classification accuracy.

Building and Deploying a Machine Learning Model with Amazon Sagemaker

This is a machine learning project involving using Amazon Web Services and Amazon Sagemaker to create and deploy a machine learning model to perform classification on health data. In this project, I utilized a Sagemaker notebook instance to preprocess a dataset full of heatlh data, and tune a pretrained XGBoost model to classify observations into various weight classes, such as 'normal weight,' 'overweight,' and 'Obese' among others. After training, I was able to deploy the model instance to a Sagemaker endpoint to be accessed within other AWS services like Lambda and AWS API Gateway.
